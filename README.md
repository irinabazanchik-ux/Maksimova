# Currency Converter (Конвертер валют)

**Автор:** [Ваше Имя и Фамилия]

## Описание
Программа позволяет конвертировать суммы между различными мировыми валютами по актуальному курсу. Все операции сохраняются в историю, которая доступна даже после перезапуска приложения.

## Как получить API-ключ
Для работы программы необходим бесплатный ключ от **ExchangeRate-API**:
1. Перейдите на сайт [exchangerate-api.com](https://www.exchangerate-api.com/).
2. Введите свой Email и нажмите "Get Free Key".
3. Подтвердите почту и скопируйте ваш API Key.
4. Вставьте его в переменную `API_KEY` в файле `main.py`.

## Примеры использования
* **Тест 1 (Обычный):** Введите `100`, выберите `USD` -> `RUB`. Нажмите кнопку. В поле результата появится сумма в рублях.
* **Тест 2 (Валидация):** Введите `-50` или слово `abc`. Программа должна выдать предупреждение об ошибке.
* **Тест 3 (История):** После успешной конвертации запись автоматически появится в таблице снизу.

## Структура проекта
* `main.py` — основной код программы.
* `conversion_history.json` — файл с историей (создается автоматически).
* `.gitignore` — исключение временных файлов и конфигов.
