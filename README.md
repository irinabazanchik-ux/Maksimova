# Currency Converter (Конвертер валют)

## Описание
Приложение для конвертации валют в реальном времени с использованием актуальных рыночных курсов. Программа ведет локальную историю операций, позволяя отслеживать прошлые расчеты.

## Как получить API-ключ
1. Перейдите на сайт [exchangerate-api.com](https://www.exchangerate-api.com/).
2. Введите Email и нажмите **"Get Free Key"**.
3. Подтвердите регистрацию в почте и скопируйте **API Key** из личного кабинета.
4. Откройте файл `main.py` и замените строку `API_KEY = "ВАШ_КЛЮЧ_API"` на полученный ключ.

## Инструкция по запуску
1. Убедитесь, что у вас установлен Python 3.7 или выше.
2. Установите зависимость `requests` (если не установлена):
```
   pip install requests

```
